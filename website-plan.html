Site Plan: La Casona Awaits You - Online Orders and More

Site Name

La Casona Awaits YouThis name is inspired on a dynamic and engaging restaurant website.

Optional Domain: https://sergiovaras.github.io/wdd231/la_casona/

Site Purpose

The website's primary objective will be to improve the restaurant's online presence, attract new customers, and facilitate interaction with existing ones.

  • To make it easier for customers to find information about the restaurant.
  • View the menu.
  • Make reservations.
  • Place orders online.
  • Improve customer experience and satisfaction.

Scenarios

  • I would like to know if it is necessary to make a reservation and how could I do so?
  • I would like to know if I can place my order online for pickup and how long it will take to have it ready?
  • Can I quickly see your daily menu or available takeout options?

Color Schema

  • Primary Color: #A0522D – Used for links, buttons, and highlight elements.
  • Accent Color: #CC5500 – Used for headings, borders, and emphasis.
  • Background: #F5F5DC – Overall page background for a clean, neutral aesthetic.

Typography

  • Playfair Display – Used for all headings to give a bold, modern look.
  • Italic – Used for body text for readability and simplicity.
